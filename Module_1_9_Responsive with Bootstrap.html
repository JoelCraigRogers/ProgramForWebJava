1.9 Bootstrap and Responsive Web Design

It is important we design with all devices in mind. Tablet/Phone/Browser/etc. Responsive web design - Create the page with all devices in mind. Bootstrap does this for us. Bootstrap uses a grid system. Will rearrange the grid elements based on the scale of the device displaying the element. This body is created inside a div class called container. This allows bootstrap to format based on the size of the screen and what it is being read on. Bootstrap uses containers and grids. Content is organized into rows. A row is a horizontal group of columns. Column classes allow you to set a size based on how many of the 12 columns it will use. If you do not specify bootstrap will size them all evenly. See Below.
